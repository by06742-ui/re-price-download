서울 빌라(연립·다세대)
실거래가 산출기

지번 또는 도로명 주소만 입력하면 대지평당가와 호별 예상 거래가를 산출합니다.

⬇️ 프로그램 다운로드 (RE_price.zip)
ZIP 다운로드 → 압축 풀기 → RE_price.exe 실행 (Windows 10/11)
  • 지번·도로명 주소로 바로 조회
  • 건축물대장 기반 사용연수·구조 자동 반영
  • 호별 대지권·평당가·예상가 산출
  • 대량 조회 결과를 2시트 엑셀로 저장
설치 안내
① 받은 RE_price.zip을 마우스 우클릭 → 압축 풀기 → ② 폴더 안 RE_price.exe 실행.
실행 시 Windows "PC를 보호했습니다" 창이 뜨면 추가 정보 → 실행을 누르면 됩니다. (서명되지 않은 개인 제작 프로그램이라 나타나는 일반적인 안내입니다.)
